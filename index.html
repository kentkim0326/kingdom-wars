1. 게임 개요

Kingdom Wars는 2D 모바일 퍼스트 스페이스 메타버스 전략 게임입니다. 실제 국가 국기를 들고 태양계 영토를 점령하세요.

핵심 컨셉

달, 화성, 목성·토성 위성에서 영토 전쟁. AI 생성 함선과 모빌슈트로 함대를 구성해 국가 간 PvP.

메타버스 포지셔닝

$KDW 코인 P2E, 크리에이터가 함선 디자인 제작·판매, 국가 자부심 중심.

타겟

13~18세 메카 팬 + 30~50대 향수 세대 + 글로벌 국가 자부심

2. 세계관

배경 스토리

서기 2387년. 지구 자원 고갈. 국가들은 국기를 들고 우주로 진출한다.

행성 진출

시즌 1: 달 (21 섹터) → 시즌 2: 화성 (35) → 목성 위성 → 토성 위성

핵심 게임플레이 루프

영토 → 유닛 → 전투

섹터 점령 → 자원 → 함선 건조 → 자동전투 → 반복

턴제 전략

비동기 전략 페이즈 + 배치 + 30~60초 시각적 자동전투

성장 경로

솔로 → 함대 → 동맹 → 길드 → 국가 황제
